# Commands

## runmqsc
MQSC commands enable you to perform administration tasks. For example, you can define, alter, or delete a local queue object.

```bash
$>runmqsc QM1
5724-H72 (C) Copyright IBM Corp. 1994, 2023.
Starting MQSC for queue manager QM1.

display queue (*)

display channel (*)

display QMGR

```
* QM1 : queue mananger
